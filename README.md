# AI-Driven Multi-Warehouse Management for WooCommerce

This repository is a **technical showcase** of a complex WordPress/WooCommerce plugin designed to digitize warehouse logistics. It demonstrates modern PHP development patterns, secure API integrations, and AI-assisted automation.

### 🚀 Key Technical Features:
- **Multi-Warehouse Stock Control:** Custom database logic to manage inventory across multiple physical locations.
- **AI Assistant (DeepSeek API):** Integrated AI to provide warehouse managers with real-time stock insights and automation.
- **Smart POS Integration:** High-performance AJAX-based checkout for in-person sales with real-time stock deduction.
- **Security-First Architecture:** Full implementation of Nonces, Capability checks, and data sanitization.

### 🛠️ Tech Stack:
- **Language:** PHP 8.1+, JavaScript (ES6)
- **Framework:** WordPress & WooCommerce Core
- **Patterns:** Object-Oriented Programming (OOP), Namespaces, Singleton-like structures.
- **APIs:** WordPress REST API & DeepSeek Chat API.
